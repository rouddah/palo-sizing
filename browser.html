Prisma Access Browser

Enterprise Browser Zero Trust · Chromium-based · Acquis via Talon (nov. 2023)

Qu'est-ce que le Prisma Access Browser ? Un navigateur d'entreprise basé sur Chromium, intégrant nativement le Zero Trust, la DLP, la prévention des menaces et les contrôles de sécurité — directement dans la couche navigateur. Il remplace le besoin d'agent VPN pour les accès web, permet de sécuriser les appareils non-gérés (BYOD, contractants) et s'intègre nativement avec Prisma Access et le Strata Cloud Manager.

Cas d'usage principaux
💻
BYOD / Appareils non-gérés
Sécurise l'accès aux apps SaaS et privées depuis les appareils personnels sans installer d'agent MDM. Pas de VPN requis — le navigateur est le périmètre de sécurité.
🤝
Contractants & Third Parties
Accès sécurisé pour prestataires et partenaires sans donner accès au réseau d'entreprise. Isolation complète et politiques de data protection renforcées.
🏢
VDI / Sessions distantes
Alternative légère aux VDI coûteux pour les travailleurs remotels ne nécessitant qu'un accès web/SaaS. Réduit significativement les coûts d'infrastructure.
🔐
Apps critiques & Data sensibles
Prévention de l'exfiltration de données (copier/coller, impression, téléchargement) sur les apps métiers critiques. Watermarking et enregistrement de session optionnel.
🌐
Remplacement VPN Web
Pour les utilisateurs n'accédant qu'à des ressources web : le PAB remplace le client VPN et offre une expérience utilisateur supérieure avec une sécurité plus granulaire.
🏭
Environnements OT / Kiosques
Sécurisation des postes partagés, kiosques industriels et environnements OT/ICS avec navigation contrôlée et isolation des sessions utilisateurs.

Points clés à retenir
Acquisition Talon (nov. 2023)
Palo Alto Networks a acquis Talon Cyber Security pour ~625 M$. La technologie est maintenant intégrée dans la plateforme PAN sous le nom "Prisma Access Browser".
Chromium — pas Chrome
Basé sur Chromium open-source (même base que Chrome/Edge), compatible avec les extensions Chrome managées. Les utilisateurs retrouvent leur environnement familier.
Zéro changement utilisateur
L'interface est identique à Chrome. Les politiques de sécurité s'appliquent de manière transparente. Pas de formation spécifique requise pour les end-users.
Intégration native PAN
Logs natifs dans SCM et Cortex XDR. Politiques partagées avec Prisma Access. Authentication unifiée avec l'IdP existant. Pas de stack sécurité séparé.
Vérification device continue
ZTNA 2.0 : contrôle de la posture de l'appareil à chaque requête (pas seulement à la connexion). Révocation automatique si la conformité change en cours de session.
Prisma Access Browser vs GlobalProtect
GP = accès réseau complet (network-centric VPN). PAB = accès web/SaaS uniquement (browser-centric Zero Trust). Les deux sont complémentaires selon les profils utilisateurs.
