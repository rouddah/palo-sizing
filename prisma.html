Ce que Pro ajoute — détail

  • Accès apps privées ZTNA : RDP, SSH et apps web privées accessibles depuis le navigateur, sans VPN et sans agent installé sur les serveurs cibles.
  • Advanced Web Protection (AWP / AXS / ABP) : inspection en temps réel du contenu et des scripts pendant l'exécution de la page ; AXS surveille les extensions de navigateur ; ABP bloque les zero-days.
  • Advanced Data Protection (Beyond) : étend les contrôles anti-fuite aux logiciels installés sur le poste (Outlook, Slack, Excel, Teams), au-delà du navigateur.
  • Advanced Identity Protection (Web PAM) : sécurise les apps hors SSO et les comptes partagés - mots de passe gérés centralement, jamais visibles par l'utilisateur.
  • ADEM RUM : mesure de l'expérience utilisateur réelle depuis le navigateur.
Connectivité apps privées (Pro)
Options de connectivité Prisma Browser Pro pour les apps privées :
  • Prisma Browser Connector : choisit automatiquement la région cloud la plus proche.
  • NGFW Connector : réutilise un firewall existant en PAN-OS ≥ 12.1.5 via Panorama.
  • Options additionnelles : Service Connection, ZTNA Connector, Colo Connect.
